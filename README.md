# Обо мне
Я аналитик данных с более чем 10-летним опытом работы с финансовыми и бизнес-данными и практическим опытом применения методов анализа данных и машинного обучения.

Основная часть моего профессионального опыта связана с анализом финансовой информации, оценкой рисков и подготовкой аналитических материалов для принятия управленческих решений. В последние годы я активно развиваюсь в области Data Science и аналитики данных: изучаю методы машинного обучения, работаю с Python и SQL, реализую проекты по анализу данных и построению моделей.

Мне особенно интересны задачи, связанные с анализом качества ML-моделей, поиском факторов, влияющих на прогноз, и формированием рекомендаций для бизнеса на основе данных. В работе я уделяю большое внимание корректной подготовке данных, их качеству и интерпретации результатов анализа.

Сейчас я ищу позицию аналитика данных или ML-аналитика в продуктовой команде, где аналитика помогает улучшать модели и находить точки роста бизнеса.
